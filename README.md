**HR Analytics Dashboard**

This project presents a comprehensive human resources analytics dashboard designed using Microsoft Power BI. It helps HR professionals and decision-makers monitor key workforce metrics, identify trends, and support strategic workforce planning through data-driven insights.

**Project Overview**

The dashboard visualizes critical HR metrics, including attrition rates, employee demographics, department-wise distributions, training impact, and hiring trends. It is built on a sample HR dataset and simulates real-world workforce analysis scenarios.

**Objectives**

- Track employee headcount, attrition, and hiring metrics
- Analyze employee distribution by department, job role, and education
- Monitor training participation and its relationship to retention
- Provide actionable insights to improve HR operations and workforce engagement

**Tools Used**

- Microsoft Excel – data cleaning and preprocessing
- Microsoft Power BI – data modeling, KPIs, and dashboard design

**Key Insights**

- Identified departments with higher attrition rates
- Demonstrated correlation between training and employee retention
- Showed workforce experience and education distribution
- Revealed trends in hiring, gender diversity, and job satisfaction

**Data Source**

The dataset used is publicly available and intended for educational and demonstration purposes. It does not represent real or proprietary organizational data.

**Folder Structure**

- /Dashboard – Contains the Power BI (.pbix) file
- /Datasets – Contains Excel or CSV files used for analysis
- README.md – Project documentation

**How to Use**

1. Clone or download this repository
2. Open the .pbix file in Power BI Desktop
3. Explore visuals, filters, and performance indicators
4. Review Excel files for raw and transformed data
